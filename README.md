# Automobile Analysis Dashboard

## Project Overview
This project provides an **end-to-end analysis of automobile data** using **Databricks**, including **data ingestion, cleaning, SQL analysis, and dashboard visualizations**. The analysis focuses on understanding trends in **sales volume, pricing, and manufacturer performance** across different countries and years.  

The project is designed for **data-driven insights** and can be extended for predictive modeling.
